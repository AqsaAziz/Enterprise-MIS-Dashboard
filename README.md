
# Enterprise MIS Dashboard — H1 2025

## Overview
Interactive 3-page MIS Dashboard built in Power BI 
tracking Budget vs Actual vs Forecast across 
6 departments for H1 2025.

## Pages
- Page 1: Executive Summary — KPI cards & variance
- Page 2: Department Analysis — drill down by dept
- Page 3: Trend Analysis — monthly trends & forecast

## Tools Used
- Power BI Desktop
- DAX Measures
- Power Query
- Excel (Data Source)

## Key Features
- Budget vs Actual vs Forecast comparison
- Department wise variance analysis
- Full year trend with forecast
- Interactive slicers for filtering
- KPI cards for quick insights

  ## Key Insights
- Total Budget H1: OMR 1,102,000
- Total Actual H1: OMR 1,100,000
- Total Variance: OMR -2,000 (Under Budget ✅)
- 6 Departments tracked across 6 months
